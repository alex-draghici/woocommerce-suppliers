=== WooCommerce Suppliers ===
Requires at least: 3.0.1
Tested up to: 3.4
Stable tag: 4.3

Adds a new "Suppliers" taxonomy to WooCommerce products.

== Description ==

Adds a new "Suppliers" taxonomy to WooCommerce products

== Installation ==

1. Upload `woocommerce-suppliers` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
